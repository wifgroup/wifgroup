Day 9: Custom Widgets & Theming

Learn how to create reusable UI components and apply a consistent style across your app. üé®

üß© Why Custom Widgets?

Flutter apps are built from widgets. If you find yourself repeating the same layout, convert it into a custom widget. This makes your code:

  • Cleaner: Reduces redundant code and makes the main widget tree easier to read.
  • Reusable: You can use the same widget in multiple places.
  • Easier to maintain: If you need to change the design, you only have to edit one file.
Custom Widget Example: ProfileCard 

class ProfileCard extends StatelessWidget {
  final String name;
  final String role;

  ProfileCard({required this.name, required this.role});

  @override
  Widget build(BuildContext context) {
    return Card(
      margin: EdgeInsets.all(8),
      child: ListTile(
        leading: Icon(Icons.person),
        title: Text(name),
        subtitle: Text(role),
      ),
    );
  }
}

Usage:

Using the Custom Widget 

ProfileCard(name: 'John Doe', role: 'Developer'),

üé® Flutter Themes ‚Äì Consistency Made Easy

Instead of styling every widget individually, use ThemeData to apply a global look and feel. This ensures your app is consistent and makes it easy to switch between light and dark themes later on.

Defining a Theme 

MaterialApp(
  theme: ThemeData(
    primarySwatch: Colors.teal,
    textTheme: TextTheme(
      bodyMedium: TextStyle(fontSize: 16, color: Colors.black87),
      titleLarge: TextStyle(fontSize: 20, fontWeight: FontWeight.bold),
    ),
  ),
  home: MyHomePage(),
)

‚úçÔ∏è Adding Custom Fonts

To use a custom font, first add the font file to a new folder in your project (e.g., assets/fonts/).

Then, update your pubspec.yaml file to include the font asset:

pubspec.yaml 

flutter:
  fonts:
    - family: Montserrat
      fonts:
        - asset: assets/fonts/Montserrat-Regular.ttf
        - asset: assets/fonts/Montserrat-Bold.ttf
          weight: 700

Finally, apply it in your theme:

Applying the Font 

theme: ThemeData(
  fontFamily: 'Montserrat',
)

‚úÖ Full Example: Custom Themed App

Here‚Äôs a complete app that combines a custom theme with a reusable custom widget.

Complete Code 

import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(
        primarySwatch: Colors.deepPurple,
        fontFamily: 'Montserrat',
        textTheme: TextTheme(
          titleLarge: TextStyle(
            fontSize: 22,
            fontWeight: FontWeight.bold,
            color: Colors.deepPurple,
          ),
          bodyMedium: TextStyle(fontSize: 16, color: Colors.black87),
        ),
      ),
      home: ThemeDemo(),
    );
  }
}

class ThemeDemo extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('Theming & Custom Widgets')),
      body: Column(
        children: [
          ProfileCard(name: 'Alice', role: 'Designer'),
          ProfileCard(name: 'Bob', role: 'Engineer'),
          Text('Custom themed text!', style: Theme.of(context).textTheme.titleLarge),
        ],
      ),
    );
  }
}

class ProfileCard extends StatelessWidget {
  final String name;
  final String role;

  ProfileCard({required this.name, required this.role});

  @override
  Widget build(BuildContext context) {
    return Card(
      margin: EdgeInsets.all(8),
      child: ListTile(
        leading: CircleAvatar(child: Icon(Icons.person)),
        title: Text(name),
        subtitle: Text(role),
      ),
    );
  }
}
