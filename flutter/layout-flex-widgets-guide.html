Day 5: Layouts & Flex Widgets

Take control of your Flutter layouts with powerful widgets like Expanded, Spacer, Wrap & Stack. üìê

‚ú® The Foundation of UI

Yesterday, we explored basic widgets like Text, Image, Row, Column, and Container. Today, we‚Äôll take a step further and learn how to control layout and spacing in Flutter using flexible widgets: Expanded, Spacer, Wrap, and Stack.

üìê 1. Expanded Widget

The Expanded widget takes available space inside Row or Column and expands a child widget to fit proportionally.

Expanded Example 

Row(
  children: [
    Expanded(child: Container(color: Colors.red, height: 50)),
    Expanded(flex: 2, child: Container(color: Colors.blue, height: 50)),
  ],
)

üëâ Here, the blue container takes double the space of the red one (flex: 2).

üìè 2. Spacer Widget

Spacer is a simpler way to add flexible empty space between widgets.

Spacer Example 

Row(
  children: [
    Icon(Icons.star, size: 40),
    Spacer(),
    Icon(Icons.favorite, size: 40),
  ],
)

üëâ The Spacer() pushes the icons to opposite ends.

üîÑ 3. Wrap Widget

Wrap automatically moves children to the next line if they don‚Äôt fit in one row.

Wrap Example 

Wrap(
  spacing: 12,
  runSpacing: 8,
  children: List.generate(
    10,
    (index) => 
        Chip(label: 
                Text('Item $index')),
  ),
)

üëâ Useful for tags, chips, grids, or responsive UI.

üé® 4. Stack Widget

Stack allows widgets to overlap each other.

Stack Example 

Stack(
  alignment: Alignment.center,
  children: [
    Container(
      width: 200, 
      height: 200, 
      color: Colors.blue.shade200),
    Positioned(
      top: 20,
      right: 20,
      child: Icon(
        Icons.star, 
        size: 40, 
        color: Colors.yellow),
    ),
    Text('Hello Stack!', 
    style: TextStyle(
      fontSize: 20, 
      fontWeight: FontWeight.bold,
      )),
  ],
)

üëâ Common use cases: profile picture overlays, banners, floating buttons, or custom designs.

üèóÔ∏è Example App ‚Äì Layout Playground

main.dart 


import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Day 5: Layouts & Flex')),
        body: Padding(
          padding: const EdgeInsets.all(16.0),
          child: Column(
            children: [
              // Expanded Example
              Row(
                children: [
                  Expanded(
                      child: Container(
                    color: Colors.red,
                    height: 40,
                  )),
                  Expanded(
                      flex: 2,
                      child: Container(
                        color: Colors.green,
                        height: 40,
                      )),
                ],
              ),
              SizedBox(height: 20),
              // Spacer Example
              Row(
                children: [
                  Icon(Icons.home, size: 40),
                  Spacer(),
                  Icon(Icons.settings, size: 40),
                ],
              ),
              SizedBox(height: 20),
              // Wrap Example
              Wrap(
                spacing: 8,
                children: List.generate(
                  6,
                  (i) => Chip(label: Text('Tag $i')),
                ),
              ),
              SizedBox(height: 20),
              // Stack Example
              Stack(
                alignment: Alignment.center,
                children: [
                  Container(
                    width: 150,
                    height: 150,
                    color: Colors.blue.shade100,
                  ),
                  Icon(
                    Icons.person,
                    size: 80,
                    color: Colors.blue.shade800,
                  ),
                  Positioned(
                    bottom: 10,
                    right: 10,
                    child: Icon(
                      Icons.check_circle,
                      color: Colors.green,
                      size: 30,
                    ),
                  ),
                ],
              ),
            ],
          ),
        ),
      ),
    );
  }
}
