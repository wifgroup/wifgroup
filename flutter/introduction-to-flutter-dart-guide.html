Day 1: Introduction to Flutter & Dart

Setup, Tools, and Getting Started

Welcome to Day 1 of our 30-Day Flutter Series! Whether you're a complete beginner or just exploring Flutter, this is the perfect starting point.

üß† What is Flutter?

Flutter is an open-source UI software development kit by Google. It allows you to create beautiful, natively compiled apps for mobile (iOS, Android), web, and desktop from a single codebase.

Key Highlights:

  • Fast Development with Hot Reload
  • Beautiful UIs with Custom Widgets
  • Same codebase across platforms
  • Powered by the Dart programming language

üí° Why Learn Flutter?

  • Single Codebase: Write once, run anywhere
  • Growing Ecosystem: Huge developer support, packages
  • Backed by Google: Used in production (e.g., Google Pay)
  • Fast & Flexible UI Design: Perfect for MVPs or complex apps

üßæ What is Dart?

Dart is the language behind Flutter. It is easy to learn, especially if you're coming from JavaScript, Java, or C-style languages.

  • Object-Oriented
  • Sound null safety
  • Great async/await support
  • Hot reload enabled

üõ†Ô∏è Install Flutter SDK

Follow these steps based on your OS:

üîó Official Flutter Installation Guide:
üëâ https://docs.flutter.dev/get-started/install

Basic Requirements:

  • Git
  • Command Line Tools
  • Internet connection

For Windows:

  • Download the SDK zip
  • Extract to a location (e.g., C:\src\flutter)
  • Add flutter\bin to your PATH
  • Run flutter doctor

For Mac:

  • brew install --cask flutter
  • flutter doctor

Ensure you install:

  • Android SDK (via Android Studio)
  • Xcode (for iOS dev)
  • Chrome (for web dev)

üì± Running Your First Flutter App

Command Line
