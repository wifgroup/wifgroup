Day 8: Lists & Grids in Flutter

Learn how to display and manage dynamic data with ListView, GridView, and performance-friendly builder patterns. üìã

üìã ListView Basics

ListView is a scrollable list of widgets. It‚Äôs perfect for showing a small, static number of items.

ListView Example 

ListView(
  children: [
    Text('Item 1'),
    Text('Item 2'),
    Text('Item 3'),
  ],
)

üß© ListTile ‚Äì A Prebuilt Row Layout

For common list rows with a title, subtitle, and leading/trailing icons, use ListTile.

ListTile Example 

ListTile(
  leading: Icon(Icons.person),
  title: Text('John Doe'),
  subtitle: Text('Software Engineer'),
  trailing: Icon(Icons.arrow_forward),
)

üîÑ ListView.builder ‚Äì Efficient Lists

When you have dynamic or large lists, use ListView.builder to only render the items that are currently visible on the screen. This is a crucial performance optimization.

ListView.builder Example 

ListView.builder(
  itemCount: 100,
  itemBuilder: (context, index) {
    return ListTile(
      leading: CircleAvatar(child: Text('${index + 1}')),
      title: Text('Item ${index + 1}'),
    );
  },
)

‚úÖ This saves memory and improves performance for long lists.

üü• GridView ‚Äì Display Items in a Grid

For layouts like galleries or product catalogs, use GridView. GridView.count is great for a fixed number of columns.

GridView.count Example 

GridView.count(
  crossAxisCount: 2,
  children: List.generate(6, (index) {
    return Card(
      margin: EdgeInsets.all(8),
      child: Center(child: Text('Item $index')),
    );
  }),
)

üîÑ GridView.builder ‚Äì Dynamic Grids

Just like ListView.builder, use GridView.builder for large dynamic data sets to boost performance.

GridView.builder Example 

GridView.builder(
  gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
    crossAxisCount: 3,
    crossAxisSpacing: 10,
    mainAxisSpacing: 10,
  ),
  itemCount: 20,
  itemBuilder: (context, index) {
    return Container(
      color: Colors.blue[100 * ((index % 8) + 1)],
      child: Center(child: Text('Item $index')),
    );
  },
)

‚úÖ Full Example: List & Grid Toggle

This example shows how to combine a StatefulWidget with a toggle button to switch between ListView and GridView views.

Complete Code 

import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: ListGridDemo(),
    );
  }
}

class ListGridDemo extends StatefulWidget {
  @override
  _ListGridDemoState createState() => _ListGridDemoState();
}

class _ListGridDemoState extends State {
  bool showGrid = false;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text(showGrid ? 'GridView Demo' : 'ListView Demo'),
        actions: [
          IconButton(
            icon: Icon(showGrid ? Icons.list : Icons.grid_on),
            onPressed: () {
              setState(() {
                showGrid = !showGrid;
              });
            },
          ),
        ],
      ),
      body: showGrid
          ? GridView.builder(
              gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
                crossAxisCount: 2,
              ),
              itemCount: 20,
              itemBuilder: (context, index) {
                return Card(
                  child: Center(child: Text('Grid $index')),
                );
              },
            )
          : ListView.builder(
              itemCount: 20,
              itemBuilder: (context, index) {
                return ListTile(
                  leading: CircleAvatar(child: Text('$index')),
                  title: Text('List Item $index'),
                  subtitle: Text('Subtitle $index'),
                );
              },
            ),
    );
  }
}
