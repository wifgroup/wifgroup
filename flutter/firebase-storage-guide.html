Day 20: Firebase Storage in Flutter

Learn to upload and manage images & files, essential for any app with media content. üåü

üß† Why Firebase Storage?

Yesterday, we mastered CRUD operations with Firestore. Today, we‚Äôll integrate **Firebase Storage** so your app can upload and fetch images or files. This is essential for apps like social media, chat, or galleries where users need to share and view media. Firebase Storage is a powerful and scalable object storage service built for app developers.

  • Store images, videos, PDFs, or any files.
  • Scales automatically with your app.
  • Integrates seamlessly with Firestore to link file URLs with data.
  • Secure file storage with Firebase Security Rules.

üì¶ Step 1: Add Dependencies

To get started, you'll need two packages: `firebase_storage` to handle the file uploads and `image_picker` to select files from the device's gallery or camera.

pubspec.yaml 

dependencies:
  flutter:
    sdk: flutter
  firebase_core: ^2.15.0
  firebase_storage: ^11.5.0
  image_picker: ^0.9.0

Run `flutter pub get` to install them.

üìå Step 2: Initialize Firebase

Make sure Firebase is properly initialized before you try to use Storage. This step was covered in Day 17. The code snippet below shows the standard initialization process in your `main()` function.

main.dart snippet 

await Firebase.initializeApp(
  options: DefaultFirebaseOptions.currentPlatform,
);

üìå Step 3: Pick an Image

The `image_picker` package provides a simple way to access the user's gallery or camera. We'll use `_picker.pickImage` to select a single image from the gallery.

Pick an image from gallery 

import 'package:image_picker/image_picker.dart';

final ImagePicker _picker = ImagePicker();
XFile? image = await _picker.pickImage(source: ImageSource.gallery);

üìù Step 4: Upload Image to Firebase Storage

Once you have the `XFile`, you can upload it to a specific path in your Firebase Storage bucket. The `putFile` method handles the upload, and `getDownloadURL` retrieves the public URL to display the image later.

Upload function 

import 'package:firebase_storage/firebase_storage.dart';
import 'dart:io';

Future uploadFile(XFile file) async {
  // Create a storage reference
  final storageRef = FirebaseStorage.instance.ref();
  final imagesRef = storageRef.child('images/${file.name}');
  
  // Upload file
  await imagesRef.putFile(File(file.path));
  
  // Get download URL
  String downloadURL = await imagesRef.getDownloadURL();
  return downloadURL;
}

üìñ Step 5: Display Uploaded Image

After uploading the image, you can store the `downloadURL` in Firestore and use it to display the image in your app. The `Image.network` widget is perfect for this.

Display image from URL 

Image.network(downloadURL)

üßë‚Äçüíª Example: Image Upload Widget

This stateful widget combines the steps above to create a functional image uploader. It allows a user to pick an image, uploads it to Firebase Storage, and then displays the uploaded image from its public URL.

Complete Code 

import 'dart:io';
import 'package:flutter/material.dart';
import 'package:image_picker/image_picker.dart';
import 'package:firebase_storage/firebase_storage.dart';

class ImageUploadDemo extends StatefulWidget {
  @override
  _ImageUploadDemoState createState() => _ImageUploadDemoState();
}

class _ImageUploadDemoState extends State {
  XFile? _image;
  String? _downloadURL;

  final ImagePicker _picker = ImagePicker();

  Future pickAndUploadImage() async {
    final XFile? pickedImage = await _picker.pickImage(source: ImageSource.gallery);
    if (pickedImage == null) return;

    setState(() => _image = pickedImage);

    final storageRef = FirebaseStorage.instance.ref().child('images/${pickedImage.name}');
    await storageRef.putFile(File(pickedImage.path));

    final url = await storageRef.getDownloadURL();
    setState(() => _downloadURL = url);
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text("Firebase Storage Upload")),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            _image != null ? Image.file(File(_image!.path), height: 150) : SizedBox(),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: pickAndUploadImage,
              child: Text("Pick & Upload Image"),
            ),
            SizedBox(height: 20),
            _downloadURL != null ? Image.network(_downloadURL!, height: 150) : SizedBox(),
          ],
        ),
      ),
    );
  }
}
