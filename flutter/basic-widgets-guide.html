Day 4: Basic Widgets in Flutter

Master the essential building blocks of any Flutter UI: Text, Image, Row, Column & Container. üß±

‚ú® The Foundation of UI

These five widgets are the foundation of Flutter layouts.

üìù 1. Text Widget

The Text widget is used to display text on the screen.

Text Widget Example 

Text(
  'Hello Flutter!',
  style: TextStyle(
    fontSize: 24,
    fontWeight: FontWeight.bold,
    color: Colors.blue,
  ),
)

‚ú® Tip: Always use TextStyle to customize fonts, colors, and sizes.

üñºÔ∏è 2. Image Widget

Display images from assets, network, or memory.

From Network:

Image.network 

Image.network(
  'https://flutter.dev/images/flutter-logo-sharing.png',
  width: 150,
)

From Assets:

Add in pubspec.yaml

pubspec.yaml 

assets:
  - assets/images/

Use in code:

Image.asset 

Image.asset('assets/images/flutter.png')

üìê 3. Row Widget

Row places widgets horizontally.

Row Example 

Row(
  mainAxisAlignment: MainAxisAlignment.spaceAround,
  children: [
    Icon(Icons.home, size: 40, color: Colors.blue),
    Icon(Icons.star, size: 40, color: Colors.orange),
    Icon(Icons.person, size: 40, color: Colors.green),
  ],
)

üìè 4. Column Widget

Column places widgets vertically.

Column Example 

Column(
  mainAxisAlignment: MainAxisAlignment.center,
  children: [
    Text('Top Widget'),
    SizedBox(height: 20),
    Text('Bottom Widget'),
  ],
)

üìå Use Row and Column together to create complex layouts.

üé® 5. Container Widget

The most versatile widget ‚Äî can be used for styling, spacing, colors, alignment, and more.

Container Example 

Container(
  padding: EdgeInsets.all(16),
  margin: EdgeInsets.all(12),
  decoration: BoxDecoration(
    color: Colors.blue.shade100,
    borderRadius: BorderRadius.circular(12),
  ),
  child: Text(
    'Inside a Container!',
    style: TextStyle(fontSize: 18),
  ),
)

üèóÔ∏è Example App ‚Äì Combining Widgets

main.dart 

import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Day 4: Basic Widgets')),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              Text('Flutter Basics', style: TextStyle(fontSize: 24)),
              SizedBox(height: 20),
              Image.network('https://flutter.dev/images/flutter-logo-sharing.png', width: 100),
              SizedBox(height: 20),
              Row(
                mainAxisAlignment: MainAxisAlignment.center,
                children: [
                  Icon(Icons.favorite, color: Colors.red),
                  SizedBox(width: 10),
                  Text('Widgets in Action!'),
                ],
              ),
            ],
          ),
        ),
      ),
    );
  }
}
