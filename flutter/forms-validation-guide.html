Day 7: Forms & Validation in Flutter

Learn how to handle user input with powerful forms, validation, and a GlobalKey. üìù

üìù TextFormField

Flutter provides TextField (basic input) and TextFormField (input with validation). Almost every real-world app needs forms: login, sign-up, profile edit, etc.

Here‚Äôs a basic example of a TextFormField:

TextFormField Example 

TextFormField(
  decoration: InputDecoration(
    labelText: 'Enter your email',
    border: OutlineInputBorder(),
  ),
)

üß© The Form Widget

Wrap multiple input fields inside a Form. The Form helps in grouping and validating them together as a single unit.

Form Widget Example 

Form(
  child: Column(
    children: [
      TextFormField(
        decoration: InputDecoration(labelText: 'Username'),
      ),
      TextFormField(
        obscureText: true,
        decoration: InputDecoration(labelText: 'Password'),
      ),
    ],
  ),
);

üîë GlobalKey for Validation

The Form needs a GlobalKey<FormState> to track its state and trigger validation and saving.

Creating the GlobalKey 

final _formKey = GlobalKey();

Attach it to the Form widget:

Attaching the Key 

Form(
  key: _formKey,
  child: Column(
    children: [
      // fields here
    ],
  ),
);

‚úÖ Full Example: Login Form with Validation

Here's a complete, working example of a login form that uses validators and a GlobalKey to validate user input on a button press.

Complete Code 

import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Form Demo',
      home: LoginScreen(),
    );
  }
}

class LoginScreen extends StatelessWidget {
  final _formKey = GlobalKey();

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('Login Form')),
      body: Padding(
        padding: EdgeInsets.all(16.0),
        child: Form(
          key: _formKey,
          child: Column(
            children: [
              TextFormField(
                decoration: InputDecoration(labelText: 'Email'),
                validator: (value) {
                  if (value == null || value.isEmpty) {
                    return 'Please enter your email';
                  }
                  if (!value.contains('@')) {
                    return 'Enter a valid email';
                  }
                  return null;
                },
              ),
              TextFormField(
                obscureText: true,
                decoration: InputDecoration(labelText: 'Password'),
                validator: (value) {
                  if (value == null || value.length < 6) {
                    return 'Password must be at least 6 characters';
                  }
                  return null;
                },
              ),
              SizedBox(height: 20),
              ElevatedButton(
                child: Text('Login'),
                onPressed: () {
                  if (_formKey.currentState!.validate()) {
                    ScaffoldMessenger.of(context).showSnackBar(
                      SnackBar(content: Text('Processing Data...')),
                    );
                  }
                },
              ),
            ],
          ),
        ),
      ),
    );
  }
}

üîç Key points:

  • Each TextFormField has a validator function that returns an error message or null.
  • FormState.validate() checks all fields at once.
  • A SnackBar shows a success message when all inputs are valid.
