Day 12: AppBar, TabBar, Drawer & BottomNavigationBar

Learn the core app structure widgets that provide navigation and a consistent user experience. üß≠

üìå AppBar

The **AppBar** sits at the top of the screen and is usually placed within a **Scaffold**. It provides a consistent place for your app‚Äôs title and primary actions.

AppBar Example 

AppBar(
  title: Text("My App"),
  centerTitle: true,
  backgroundColor: Colors.deepPurple,
  actions: [
    IconButton(icon: Icon(Icons.search), onPressed: () {}),
    IconButton(icon: Icon(Icons.more_vert), onPressed: () {}),
  ],
)

üóÇÔ∏è TabBar + TabBarView

A **TabBar** allows you to switch between different screens or content sections. It's often used with **DefaultTabController** and **TabBarView** to handle the state and display of each tab.

TabBar Example 

DefaultTabController(
  length: 3,
  child: Scaffold(
    appBar: AppBar(
      title: Text("TabBar Example"),
      bottom: TabBar(
        tabs: [
          Tab(icon: Icon(Icons.home), text: "Home"),
          Tab(icon: Icon(Icons.star), text: "Favorites"),
          Tab(icon: Icon(Icons.person), text: "Profile"),
        ],
      ),
    ),
    body: TabBarView(
      children: [
        Center(child: Text("Home Content")),
        Center(child: Text("Favorites Content")),
        Center(child: Text("Profile Content")),
      ],
    ),
  ),
);

üìÇ Drawer

The **Drawer** is a side panel that slides out from the edge of the screen, providing a place for your app's main navigation menu or less-frequently used options.

Drawer Example 

Scaffold(
  appBar: AppBar(title: Text("Drawer Example")),
  drawer: Drawer(
    child: ListView(
      padding: EdgeInsets.zero,
      children: [
        DrawerHeader(
          decoration: BoxDecoration(color: Colors.blue),
          child: Text("My Drawer", style: TextStyle(color: Colors.white)),
        ),
        ListTile(
          leading: Icon(Icons.home),
          title: Text("Home"),
          onTap: () {},
        ),
        ListTile(
          leading: Icon(Icons.settings),
          title: Text("Settings"),
          onTap: () {},
        ),
      ],
    ),
  ),
  body: Center(child: Text("Main Screen")),
);

üìç BottomNavigationBar

The **BottomNavigationBar** is a persistent widget at the bottom of the screen, used for navigating between a small number of top-level destinations (typically 3-5). It requires a **StatefulWidget** to manage the selected index.

BottomNavigationBar Example 

class BottomNavDemo extends StatefulWidget {
  @override
  _BottomNavDemoState createState() => _BottomNavDemoState();
}

class _BottomNavDemoState extends State {
  int _currentIndex = 0;

  final List _screens = [
    Center(child: Text("Home Screen")),
    Center(child: Text("Search Screen")),
    Center(child: Text("Profile Screen")),
  ];

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text("BottomNavigationBar")),
      body: _screens[_currentIndex],
      bottomNavigationBar: BottomNavigationBar(
        currentIndex: _currentIndex,
        onTap: (index) {
          setState(() {
            _currentIndex = index;
          });
        },
        items: [
          BottomNavigationBarItem(icon: Icon(Icons.home), label: "Home"),
          BottomNavigationBarItem(icon: Icon(Icons.search), label: "Search"),
          BottomNavigationBarItem(icon: Icon(Icons.person), label: "Profile"),
        ],
      ),
    );
  }
}

üßë‚Äçüíª Example App: Combining All

This comprehensive example shows how to combine an **AppBar**, a **TabBar**, a **Drawer**, and a **BottomNavigationBar** within a single **Scaffold** to create a complete app navigation experience.

Complete Code 

import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(home: MainApp());
  }
}

class MainApp extends StatefulWidget {
  @override
  _MainAppState createState() => _MainAppState();
}

class _MainAppState extends State {
  int _currentIndex = 0;

  final List _screens = [
    Center(child: Text("üè† Home")),
    Center(child: Text("‚≠ê Favorites")),
    Center(child: Text("üë§ Profile")),
  ];

  @override
  Widget build(BuildContext context) {
    return DefaultTabController(
      length: 3,
      child: Scaffold(
        appBar: AppBar(
          title: Text("Day 12: Navigation Widgets"),
          bottom: TabBar(
            tabs: [
              Tab(icon: Icon(Icons.home), text: "Tab 1"),
              Tab(icon: Icon(Icons.star), text: "Tab 2"),
              Tab(icon: Icon(Icons.person), text: "Tab 3"),
            ],
          ),
        ),
        drawer: Drawer(
          child: ListView(
            children: [
              DrawerHeader(
                decoration: BoxDecoration(color: Colors.blue),
                child: Text("Flutter Drawer", style: TextStyle(color: Colors.white)),
              ),
              ListTile(title: Text("Home"), leading: Icon(Icons.home)),
              ListTile(title: Text("Settings"), leading: Icon(Icons.settings)),
            ],
          ),
        ),
        body: TabBarView(children: _screens),
        bottomNavigationBar: BottomNavigationBar(
          currentIndex: _currentIndex,
          onTap: (index) => setState(() => _currentIndex = index),
          items: [
            BottomNavigationBarItem(icon: Icon(Icons.home), label: "Home"),
            BottomNavigationBarItem(icon: Icon(Icons.search), label: "Search"),
            BottomNavigationBarItem(icon: Icon(Icons.person), label: "Profile"),
          ],
        ),
      ),
    );
  }
}
