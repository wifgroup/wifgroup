Day 18: Firebase Authentication in Flutter

Learn to implement user sign-up, sign-in, and session management with Firebase Auth. üîë

üß† Why Firebase Auth?

Yesterday, we connected our Flutter app to Firebase and set up Firestore. Today, we‚Äôll focus on **Firebase Authentication**, which is a comprehensive, secure, and easy-to-implement solution for managing users in your app. It handles everything from user registration and login to maintaining sessions across app launches.

Firebase Auth supports a variety of sign-in methods, including email/password, Google, Facebook, and phone numbers. For this lesson, we will focus on the most common method: **email and password**.

üì¶ Step 1: Add Dependencies

First, ensure your `pubspec.yaml` file includes the necessary Firebase dependencies. The `firebase_auth` package is what we will use to handle the authentication logic.

pubspec.yaml 

dependencies:
  flutter:
    sdk: flutter
  firebase_auth: ^5.1.0
  firebase_core: ^2.15.0

Run `flutter pub get` in your terminal to install the packages.

üìå Step 2: Initialize Firebase

As covered in Day 17, you must initialize Firebase before using any of its services. This is typically done in your `main()` function.

main.dart snippet 

await Firebase.initializeApp(
  options: DefaultFirebaseOptions.currentPlatform,
);

üìù Step 3: Sign Up a User

To create a new user with an email and password, use `createUserWithEmailAndPassword`. This function is asynchronous and returns a `UserCredential` object upon success. It‚Äôs important to wrap this in a `try-catch` block to handle potential errors gracefully.

Sign Up Function 

import 'package:firebase_auth/firebase_auth.dart';

final FirebaseAuth _auth = FirebaseAuth.instance;

Future signUp(String email, String password) async {
  try {
    UserCredential userCredential = await _auth.createUserWithEmailAndPassword(
      email: email,
      password: password,
    );
    print("User Signed Up: ${userCredential.user?.email}");
  } on FirebaseAuthException catch (e) {
    if (e.code == 'weak-password') {
      print("The password is too weak.");
    } else if (e.code == 'email-already-in-use') {
      print("The account already exists for that email.");
    }
  } catch (e) {
    print(e);
  }
}

üîë Step 4: Sign In a User

Once a user has an account, they can sign in using `signInWithEmailAndPassword`. This function also returns a `UserCredential` and should be handled with a `try-catch` block to manage errors like a wrong password or a non-existent user.

Sign In Function 

Future signIn(String email, String password) async {
  try {
    UserCredential userCredential = await _auth.signInWithEmailAndPassword(
      email: email,
      password: password,
    );
    print("User Signed In: ${userCredential.user?.email}");
  } on FirebaseAuthException catch (e) {
    if (e.code == 'user-not-found') {
      print("No user found for that email.");
    } else if (e.code == 'wrong-password') {
      print("Wrong password provided.");
    }
  }
}

üìå Step 5: Sign Out a User

To sign out the currently logged-in user, simply call the `signOut()` method on your `FirebaseAuth` instance. This clears the user's session.

Sign Out Function 

Future signOut() async {
  await _auth.signOut();
  print("User Signed Out");
}

üßë‚Äçüíª Example: Simple Auth Screen

This simple screen combines all the authentication functions we've discussed, allowing users to sign up, sign in, and sign out with a basic UI.

Complete Code 

import 'package:flutter/material.dart';
import 'package:firebase_auth/firebase_auth.dart';

class AuthScreen extends StatefulWidget {
  @override
  _AuthScreenState createState() => _AuthScreenState();
}

class _AuthScreenState extends State {
  final _auth = FirebaseAuth.instance;
  String email = '';
  String password = '';

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text("Firebase Auth")),
      body: Padding(
        padding: EdgeInsets.all(16),
        child: Column(
          children: [
            TextField(
              onChanged: (value) => email = value,
              decoration: InputDecoration(labelText: "Email"),
            ),
            TextField(
              onChanged: (value) => password = value,
              decoration: InputDecoration(labelText: "Password"),
              obscureText: true,
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () async {
                await _auth.createUserWithEmailAndPassword(
                    email: email, password: password);
              },
              child: Text("Sign Up"),
            ),
            ElevatedButton(
              onPressed: () async {
                await _auth.signInWithEmailAndPassword(
                    email: email, password: password);
              },
              child: Text("Sign In"),
            ),
            ElevatedButton(
              onPressed: () async {
                await _auth.signOut();
              },
              child: Text("Sign Out"),
            ),
          ],
        ),
      ),
    );
  }
}
